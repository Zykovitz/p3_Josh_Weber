This is a page about a sketch that I made!

This sketch was made for a non-electrial powered machine that I had to build for a science project. The objective of the project was to make a machine move the most that it can, but not be powered. I used rubber bands, discs, and a baseplate of cardboard to move my machine. I didn't win, but I was able to go pretty far, considering that I had no engine or electricty.
